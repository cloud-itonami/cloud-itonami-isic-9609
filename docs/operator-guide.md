# Operator Guide

## First Deployment

1. Register the operator's license, jurisdiction and responsible staff.
2. Import historical member/client/customer records and counterparties.
3. Run read-only validation of existing records against this blueprint's
   contracts.
4. Configure the Personal Service Governor's hold/escalation policy.
5. Publish a dry-run operation and audit export.

## Minimum Production Controls

- verification/citation required before any customer-facing determination
- finalizing a personal referral or introduction always requires a human sign-off
- audit export for every hold, approval and service/publication action
- backup manual process for governor/system outage

## Certification

Certified operators must prove record integrity, governor independence,
evidence-backed reporting and human review for every high-stakes action.
